2 Definitions,theorems are written with div code-blocks

  • #def-, #thm-line are used in div-ID to cross reference Definitions or Theorems
  • tex-blocks are used to display equations within div
  • can fence the def-thm with div-block with css-style border: 1pt black solid; padding: 6pt
  • .proof is a class with the div block

Definition 1 An \(event\) is any collection of possible outcomes of an experiment, that is, any subset of S (including S itself).

An $event$ is any collection of possible outcomes of an experiment,

Theorem 1 If P is a probability function and A is any set in \(\mathscr{B}\) then,

  1. \(P(0) = 0,\) where \(0\) is an empty set;
  2. \(P(A) \le 1\);
  3. \(P(A^c) = 1 - P(A)\).
::: {#thm-line}
If P is a probability function and A is any set in $\mathscr{B}$ then,

a.  $P(0) = 0,$ where $0$ is an empty set;
b.  $P(A) \le 1$;
c.  $P(A^c) = 1 - P(A)$.
:::

4 Equations in a TeX block

Use \begin{align*} block for equations (see snippet ali*) \[\begin{align*} & \int_0^2 (3x^2+1) dx \\ &= {x^3} + x {\Big|}_0^2 \\ &= (2^3 +2) - (0^3 + 0) \\ &= 10 -(0) = 10 \qquad \blacksquare \end{align*}\]

\begin{align*}
  & \int_0^2 (3x^2+1) dx \\
  &= {x^3} + x {\Big|}_0^2  \\
  &= (2^3 +2) - (0^3 + 0) \\
  &= 10 -(0) = 10 \qquad \blacksquare
\end{align*}
